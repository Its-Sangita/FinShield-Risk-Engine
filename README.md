# FinShield-Risk-Engine
An automated Credit Risk Assessment engine built in Python to assist Micro-finance NGOs in evaluating loan eligibility for underserved communities using weighted scoring algorithms and non-traditional data points.

FinShield: Micro-Lending Risk Analysis Engine 🛡️💰
FinShield is a specialized financial technology (FinTech) tool designed to bridge the gap between traditional banking logic and social impact initiatives. Developed specifically to address the challenges faced by micro-finance NGOs, this engine provides a robust framework for assessing creditworthiness in populations without traditional credit histories.

🌟 Overview
In many underserved communities, traditional credit scores do not exist. FinShield uses Python-based logic to evaluate "Non-Traditional Data Points"—such as business stability and disposable income ratios—to help NGOs lend safely and sustainably.

This project aligns with the mission of providing technology solutions for Social Good, mirroring the objectives of the JPMorganChase 'Force for Good' program.

🛠️ Key Features
Weighted Risk Scoring: Implements a multi-factor algorithm that weighs three critical financial pillars:

Debt-to-Income (40%): Evaluates the "buffer" between earnings and survival expenses.

Stability Factor (30%): Analyzes the longevity of the applicant's business or trade.

Financial Reserves (30%): Checks for emergency savings relative to monthly costs.

Automated Decisioning: Instantly categorizes applicants into Low, Medium, or High risk tiers.

Scalable Architecture: Built using Object-Oriented Programming (OOP) to handle large datasets with O(n) time complexity.

🚀 Tech Stack & Logic
Language: Python 3.x

Core Concepts: Classes & Objects (OOP), Data Structures (Lists/Dictionaries), Logic Flow.

Methodology: Follows Agile development principles for modular and readable code.
